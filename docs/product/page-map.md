## 路由与权限原则

- `[id]` 页面必须同时校验登录身份和交换成员资格；仅凭 URL 不可读取内容。
- 私人表达、整理和预览页面只返回当前用户自己的材料。
- 信件阅读页仅在送达后向收信人开放；托管状态不能通过接口或页面源数据提前泄露正文。
- 交汇与记忆页面仅在生成条件满足后向双方开放。
- 一段人生的私人时间线只返回当前用户自己的记录；对方不得读取记录数量、具体日期或媒体元数据。
- 双方独立表达模式中，个人源稿提交后先托管；P29 只有双方提交并锁定公开快照后才能访问。
- 登录注册完成后回到被暂存的业务目标，不统一跳转首页。
