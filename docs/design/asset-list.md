# Demo 资产清单

## 已实现原创资产

| 资产 | 实现方式 | 用途 |
|---|---|---|
| 雾感入口场景 | CSS 渐变、山丘、树与路径 | 品牌、登录氛围 |
| 四种记忆信使 | CSS 不规则形状 | 陪伴、送信、信使选择 |
| 三套叙事插图 | CSS 场景构成 | 信件封面、预览、记忆碎片 |
| 共同记忆树 | CSS 树冠、双树干、根系和节点 | 记忆树页面 |
| 语音波形 | CSS 动画 | 模拟录音状态 |
| 通用图标 | `lucide-react` | 导航、按钮和状态 |

## 未来替换

- 正式品牌 Logo、插画、信使动作帧和专属表情需单独原创设计与版权确认。
- 真实上传图片使用浏览器临时预览；刷新后的文件持久化需未来对象存储支持。
- TTS、录音和 AI 生成资源当前均为交互模拟，不包含收费 API 产物。
